# Release Notes - v1.2.0

## New Features

### UEBA (User Entity Behavior Analytics)
- Behavioral baseline creation from 30-day audit log history
- Anomaly detection: geographic, temporal, operational, and IP anomalies
- Risk scoring (0-100) with actionable recommendations
- Automatic alert generation for high-risk activity (score >= 70)
- UEBA Dashboard page with baseline listing and risk detail panel

### Case Management (Incident Response)
- Full incident lifecycle: new → investigating → contained → resolved → closed
- Timeline tracking with user attribution
- Evidence management with chain of custody
- Incident assignment and statistics dashboard
- Playbook execution from incident detail view

### Automated Playbooks
- Compromised Account Response (6 steps, manual approval for containment)
- Phishing Email Response (5 steps, auto-execute containment)
- Privileged Access Abuse (4 steps, manual approval for admin disable)
- Database-backed execution tracking with status and results

### Threat Intelligence
- Multi-provider IOC enrichment (VirusTotal, AbuseIPDB, Shodan, IPQualityScore)
- Single and bulk IP/domain/hash enrichment
- Saved IOCs with CRUD tracking
- Risk scoring and level classification
- 1-hour caching with automatic cleanup

## API Endpoints
- `GET/POST /api/ueba/*` - UEBA baseline and risk endpoints
- `GET/POST/PUT /api/incidents/*` - Full incident management
- `GET/POST /api/threat-intel/*` - IOC enrichment
- `GET/POST/DELETE /api/threat-intel/saved/*` - Saved IOCs

## Database Migrations
- `007_add_ueba_incidents_playbooks.sql` - UEBA baselines, incidents, timeline, evidence, playbook executions
- `008_add_saved_iocs.sql` - Saved IOCs table

## Required Environment Variables
```bash
VIRUSTOTAL_API_KEY=         # Optional
ABUSEIPDB_API_KEY=          # Optional
SHODAN_API_KEY=             # Optional

```

## Deployment
1. Apply migrations: `007_add_ueba_incidents_playbooks.sql` and `008_add_saved_iocs.sql`
2. Optionally configure threat intel API keys in `.env`
3. Restart API and frontend services
